# Real Code Glyph Documentation

This folder contains comprehensive documentation for the font-manipulation research project. Start here for a guided tour of the repository, then drill into the focused guides for backend, frontend, CLI tooling, fonts, and verification assets.

## Documentation Map

- [`repository-overview.md`](repository-overview.md) — High-level layout of the repo with descriptions for every directory and key file.
- [`backend.md`](backend.md) — Flask API internals, manipulators, logging, and outputs.
- [`frontend.md`](frontend.md) — Vite/React application structure and integration points.
- [`cli-and-scripts.md`](cli-and-scripts.md) — Standalone Python utilities, demos, and how to run them safely.
- [`fonts-and-artifacts.md`](fonts-and-artifacts.md) — Reference for bundled fonts, generated PDFs, and verification documents.
- [`testing-and-verification.md`](testing-and-verification.md) — Automated tests, manual verification guides, and recommended workflows.

> **Tip:** Keep this documentation in sync with code changes; each guide contains an “Update Checklist” section to remind contributors what needs refreshes when behaviour changes.
